# metaweb-script-playground

Playground for testing the META Script implementation.

Using [metaweb-script](https://github.com/meta-web/metaweb-script) package.

## Usage

**Build** and open `index.html` file

```bash
npm run build
```

or **start lite server**

```
npm run start
```
